# Portfolio — Benjamin Balayre

Site CV/portfolio personnel bilingue (FR/EN) de Benjamin Balayre, déployé sur Vercel.

## Stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS + DaisyUI · Framer Motion · next-i18next

## Développement

```bash
yarn dev      # http://localhost:3000
yarn build    # Build de production
yarn lint     # ESLint
```

Node.js >= 24 requis.
